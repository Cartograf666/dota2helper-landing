# chereshna-net (rebuild)

Статическая вёрстка, визуально повторяющая `chereshna.net`, с адаптивностью под мобильные брейкпоинты.

## Запуск

Проще всего открыть `index.html` в браузере.

Если нужен локальный сервер (чтобы всё было 1:1 как на хостинге):

```bash
python3 -m http.server 5173
```

Открыть `http://localhost:5173`.
